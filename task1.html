Introduction to Data modeling

What you should know about data modeling

Main concepts

  1. Enterprise

    2. Entirety of an organization's technical infrastructure

  2. Models and diagrams

    3. Model: A scaled-down, non-functional representation of an object, system, or idea.

  3. Diagram

    4. A two- dimensional visual representation that is often abstracted or symbolic.

  4. Master Data Managment (MDM)

    5. This data describes an entity that is a person, place or thing. A customer is a person, and provbably the most reference example of a master data, think customer relational managment system or CMS.
    Generally the attributes of things described as master data don't change or don't change often; think of a person's eye color, or a building, street address, or a sensors serial number.

  5. Transactional data

    6. when a customer make a purchase from a business who is a master data. Or a guest who is a Master data entity checks into a hotel, which is also a Master data entity. The data describing that purchase or the check in process is called transactional data.

  6. Lookup data

    7. lookup data is just as it sounds, a lot of times is abstracted into codes, so we need to be able to look up that value of that code right and know what it means.
    Lets say we have the letter W to another entity, say the phone type entity, that also has A-W code as its identifier; but it alsohas other attributes, described the phone numer type like description column, where the value is work phone. So now we know that when we see A-W we can link it back to this other entity and know that it means work phone. The more normalized your data model, the more lookup entities you will likely see.

  7. Metadata

    8. this is data about the other three types of data; for example, an entity call person, now that person might have an attribute called eyes color and if that person has green eyes, then it would be a string with the word green, and if they have an attribute for age, that might be a tiny image for their age.
